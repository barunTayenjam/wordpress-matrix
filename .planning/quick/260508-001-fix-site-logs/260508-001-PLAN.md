# Quick Task: Fix site logs command

**ID:** 260508-001
**Description:** Fix `./matrix logs <site>` hanging forever due to `docker logs -f` (follow mode), which prevents the frontend API from returning site logs.

**Action:** Replace all `logs -f` invocations in `show_logs()` with `logs --tail=100` so the command outputs recent logs and exits.
**Files:** `matrix` (show_logs function)
**Verify:** `./matrix logs su21` exits cleanly within 5 seconds
**Done:** Outputs last 100 lines of container logs and returns

**Files:** `frontend/app.js`
