# Plan: Make ./matrix respect htaccess for WordPress sites

## Tasks

### Task 1: Enable htaccess support for existing site

**Files:** 
- `./matrix` (main script with web-server command)

**Action:**
Run `./matrix web-server apache <site-name>` for each existing WordPress site that needs htaccess support.

**Verify:**
- Site uses Apache image: `docker ps | grep wp_<site>`
- .htaccess file is honored in redirects

**Done:**
htaccess support enabled for the specified WordPress site.
